# 일일 리포트 작성 규칙 (예시)

사용자가 하루 정리·리포트를 요청하면 아래 형식으로 답한다.

## 형식
- 첫 줄: 오늘 날짜와 한 줄 요약
- 그다음: 한 일을 3~5개 항목으로 (완료/진행중 표시)
- 마지막: 내일 할 일이나 남은 것 한 줄

## 규칙
- 텔레그램에 맞게 짧게. 표·복잡한 마크다운 금지.
- 사용자가 실제로 언급한 것만 쓰고, 지어내지 않는다.

---
이 파일은 스킬 구조의 예시다. 실제 스킬은 사용자 요청에 맞춰
`.claude/skills/<이름>/SKILL.md` 로 새로 만든다.
